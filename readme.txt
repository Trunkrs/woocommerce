Add the excellent consumer focused Trunkrs shipping to your WooCommerce store.

== Description ==

The Trunkrs for WooCommerce plugin will let you integrate the best consumer focused carrier service to your WooCommerce store.

In less then 10 minutes you can start shipping with our best rated shipping service.

= Main features =
- Automatically announce shipments with Trunkrs
- Show users when they will receive their packages even before they order
- Add the Trunkrs track & trace to the checkout confirmation email
- Print shipping labels right from the admin dashboard
- Manage shipment states right from the admin dashboard

== Frequently Asked Questions ==

= What do I need to get started? =

You will need to be a customer of Trunkrs and have been registered by our customer success department on the Trunkrs platform.
Not yet a customer? No problem, [https://trunkrs.nl](go to our website) to reach out to us to get started!

= I don't have a Trunkrs platform account, how do I get it? =

Reach out to your customer success manager or account manager to get started, they will be happy to send you an invite!

== Changelog ==

= 1.2.0 =
This is a big release with lots of new and cool features you will love!
We added the following features:
- Support for order filter rules that let you choose which orders should go to Trunkrs.
- Support for Tyche Delivery Date plugin for custom delivery dates.
- Support for WooCommerce Shipment Tracking extension.
- Support for WooCommerce Subscriptions extension.

= 1.1.0 =
This release contains more settings to adjust the way we show up in your cart and checkout area.
It's now also possible to redirect all orders to the Trunkrs shipping service.

= 1.0.0 =
The first release of the plugin. All of the basic functionality has been implemented.
